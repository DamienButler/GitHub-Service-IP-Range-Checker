About This Tool

This tool queries the GitHub Meta API endpoint (https://api.github.com/meta) to verify whether an IP address or CIDR range belongs to GitHub's infrastructure. It identifies which GitHub service(s) the IP is associated with, including Actions, Pages, Copilot, Codespaces, and more.

Security & Privacy

This application runs entirely client-side in your browser. No data is sent to any backend server. All IP lookups and CIDR calculations are performed locally using JavaScript. The only external request made is to GitHub's public Meta API. Your searched IP addresses are never logged or transmitted anywhere.

Daily Monitoring

A GitHub Actions workflow runs daily to monitor the Meta API endpoint for changes. When new IP ranges are added or existing ranges are modified, the updates are recorded and displayed in the Recent Updates tab. This provides a historical audit trail of GitHub's infrastructure IP changes.
